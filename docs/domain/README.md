# ドメイン知識ドキュメント

このディレクトリには、ユビキタス言語（Ubiquitous Language）に関するドメイン知識がまとめられています。

## 目次

### 1. [概要 (overview.md)](./overview.md)
ユビキタス言語の定義、主要な特徴、重要性について説明しています。
- ユビキタス言語とは何か
- なぜ重要なのか
- DDDにおける位置づけ
- 境界づけられたコンテキストとの関係

**対象読者**: ユビキタス言語について初めて学ぶ方、概念を理解したい方

### 2. [ベストプラクティス (best-practices.md)](./best-practices.md)
ユビキタス言語を効果的に構築・運用するための実践的な方法論を紹介しています。
- 言語の構築方法
- ネーミング規約
- 実装パターン
- よくある落とし穴と対策

**対象読者**: 実際にユビキタス言語を導入・運用する開発チーム

### 3. [実践事例 (case-studies.md)](./case-studies.md)
実際の企業やプロジェクトでのユビキタス言語の活用事例を紹介しています。
- 国内企業の実践例（Leaner、LegalOn Technologies、福郎ラボ、Timee、KAKEHASHIなど）
- 海外の事例（製薬ネットワークシステム）
- 成功要因の分析
- AI時代における新しい視点

**対象読者**: 他社の事例から学びたい方、導入の参考にしたい方

### 4. [管理方法とツール (tools-and-management.md)](./tools-and-management.md)
ユビキタス言語を管理するためのツールとプロセスについて詳しく解説しています。
- 管理ツールの比較（Notion、GitHub、Confluenceなど）
- 管理プロセス（提案、更新、廃止）
- 自動化とツール連携
- 測定と改善

**対象読者**: ユビキタス言語の管理体制を構築したい方

### 5. [参考資料 (references.md)](./references.md)
ユビキタス言語に関する書籍、記事、オンラインリソースをまとめています。
- 主要な書籍
- 権威的なオンラインリソース
- 日本語の実践記事
- 学習パス

**対象読者**: さらに深く学びたい方、最新情報をキャッチアップしたい方

## 推奨される読み進め方

### 🔰 初めての方
1. **概要** → 基本概念を理解
2. **実践事例** → 具体的なイメージを持つ
3. **ベストプラクティス** → 実践方法を学ぶ

### 🚀 導入を検討している方
1. **実践事例** → 他社の事例を参考に
2. **管理方法とツール** → 管理体制を検討
3. **ベストプラクティス** → 実装の詳細を理解
4. **参考資料** → さらなる情報収集

### 💪 すでに実践している方
1. **ベストプラクティス** → 改善点を探す
2. **実践事例** → 他社との比較
3. **管理方法とツール** → 効率化のヒント
4. **参考資料** → 最新トレンドのキャッチアップ
